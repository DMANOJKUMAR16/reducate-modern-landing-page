# Reducate. — Modern Education Landing Page

A modern, responsive, and visually engaging education landing page inspired by the JG University website redesign assignment.

Built using Next.js, Tailwind CSS, and Framer Motion with a strong focus on modern UI/UX, responsiveness, smooth animations, and clean component architecture.

---

## 🚀 Live Demo

Live Site: https://reducate-modern-landing-page.vercel.app

---

## 📌 Features

- Modern responsive landing page
- Premium UI/UX design
- Fully mobile responsive
- Smooth scroll navigation
- Interactive hover effects
- Scroll animations using Framer Motion
- Clean and reusable component structure
- Optimized layout and typography
- Sticky responsive navbar
- Modern cards and CTA sections

---

## 🛠️ Tech Stack

- Next.js

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/DMANOJKUMAR16/reducate-modern-landing-page.git
```

Navigate into the project folder:

```bash
cd reducate-modern-landing-page
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
